Data Visualization Project

Interactive Dashboard (Tableau)

This project focuses on visualizing and analyzing a sample weather dataset using Tableau. The dataset contains key weather parameters such as temperature, humidity, wind speed, precipitation, cloud cover, atmospheric pressure, UV index, season, visibility, location type, and weather condition.

Objectives
  • Explore patterns and relationships between different weather factors.
  • Analyze how weather conditions vary across different seasons and locations (inland, coastal, mountain).
  • Visualize temperature, humidity, and precipitation trends.
  • Identify correlations between UV index, cloud cover, and atmospheric pressure.
  • Create interactive dashboards for better data-driven insights.
Key Features
  • Dynamic Filters and Slicers
  • Drill-down and Interactive Charts
  • Responsive Layout for Web Publishing
  • Color-coded Visual Insights
Tools Used
  • Tableau Public
  • Data Cleaning in Excel/SQL
  • Dashboard Design Principles
