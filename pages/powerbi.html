Data Visualization Project

Interactive Dashboard (Power BI)

This Tableau project focuses on analyzing agricultural operations data collected by a Ukrainian agricultural company. The dataset spans several Excel sheets, each covering different aspects of field work, resource usage, and planning. All source data is in the Ukrainian language, reflecting the local operational context.

ðŸ“‚ Dataset Structure:

The Excel file contains multiple sheets including:

  1. Work Shifts and Document Records
    Details of work documents with document numbers, dates, and shift periods.

  2. Field Operations and Work Areas
    Data about field operations such as cluster (geographical region), crop type, type of work performed, operational stage, tractor used, and the area covered (in hectares).

  3. Planned Agricultural Activities
    Planning data including clusters, crop types, fertilizers/materials to be used (TMC), planned work area, start and end dates, and plan type (e.g., current plan).

  4. Material Usage per Document Line
    Actual consumption of materials (fertilizers, etc.) by document and document line, showing quantity used and material type.

  5. Tractor Assignments
    Data about tractor allocations to specific clusters with assignment dates.
